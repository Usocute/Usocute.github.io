# About this website

  • Miraitowa: Miraitowa (发音为 mee-rah-e-toh-wa), 基于日语单词 mirai(未来) 和 towa(永恒) 的组合，意为 永远期待着明亮的未来 。
  • 草萤非火：取自唐代白居易的《放言五首・其一》中的 " 草萤有耀终非火，荷露虽团岂是珠。 ", 意为草丛间的萤虫终究不是火；荷叶上的露水也不是珍珠，然而，它们偏能以闪光、晶莹的外观炫人，人们又往往为假象所蒙蔽。(告诉我们要善于透过现象看出本质)
